** 电压和电流
   - 电荷是双极性的,分正和负
   - 电荷是离散量，是1.6022x10-19 C 的整数倍
   - 电现象归结为电荷的分离和电荷的运动

   *电荷的分离引起电压，电荷的运动引起电流*
** 功率
   \[ $$ p = dw/dt  = (dw/dq) / (dq/dt) = u * i \]

   - 电路分析基于电压和电流变量
   - 电压是由电荷分离产生的单位电荷的能量
   - 电流是电荷流动的速率
   - 理想基本电路元件是不能被分解的双端元件，它能够用端电压和电流进数学描述
   - 无源符号约定:当通过元件电流的参考方向与元件上电压降的参考方向一致，表示元件端电压和电流的相关表达式使用正号
   - 功率是单位时间的能量，功率等于端电压和电流的乘积

* 电路元件
  由于电功率的产生和分配方式,容易确定的是电压而不是电流
  电源的重要特性是既能吸收也能释放电功率
  有源元件: 可以产生电能量的器件
  无源元件: 不能产生电能量的器件
  - 理想电压源保持规定的电压，与流过器件的电流无关。理想电流源保持规定的电流，与器件上的电压无关。电压源和电流源或者是独立源，即不受电路中任何其他电压或电流的影响，或者是非独立源，即由电路中其他电流或电压确定。
  - 电阻的电压和电流相互成比例。电阻中与电压和电流关联的比例常数称为电阻，它的单位是欧姆
  - 欧姆定律 \[ $$ u = iR \]
  - 电阻功率 \[ $$ u = i^{2}R\] \[ $$ u = u^{2}/R\]
  - 电路可以通过节点和闭合路径描述。节点是两个或多个电路元件的结合点。当正好两个元件连接构成节点时，就称为串联。闭合路径是沿着连接元件形成的回路，起点和终点是相同的节点，而且遇到的每个中间节点不超过一次
  - *基尔霍夫电流定律* - 在电路中，任何节点上的所有电流的代数和等于0
  - *基尔霍夫电压定律* - 在电路中，环绕任何闭合路径的所有电压的代数和等于0

* 简单电阻电路
  - 串联电阻等效 \[ $$ R_{eq} = \sum_{i=1}^{k} R_i\]
  - 并联电阻等效 \[ $$ 1/_R_{eq} = \sum_{i=1}^{k} 1/_R_i\]
  - 分压法:从串联电阻中选中的一个电阻上的电压降
  - 分流法:从并联电阻中选中的一个电阻上的电流
  - 电压表测量电压必须与被测电压的电路元件并联
  - 电流表测量电流必须与被测电流的电路元件串联
  - 惠斯通电桥电路用于精确地测量电阻的数值。其中有四个电阻，一个直流电压源和一个检流计，当其处于平衡状态，检流计的读数为0
  - 一个电路的三个电阴如果连成\[\bigtriangleup\]结构(或\[\pi\])，则可以变换成等效电路的形式，等效电路中的三个电阻是Y连接(或T连接)

* 电路分析法
  | 名称     | 定义                                   |
  |----------+----------------------------------------|
  | 节点     | 两个或更多电路元件的连接点             |
  | 基本节点 | 三个或更多电路元件的连接点             |
  | 路径     | 基本元件相连的踪迹，元件不能出现两次   |
  | 支路     | 连接两个节点的路径                     |
  | 基本支路 | 连接两个基本节点的路径，不通过基本节点 |
  | 回路     | 终点和起点是同一节点的路径             |
  | 网孔     | 没有包围其他回路的回路                 |
  | 平面电路 | 画在平面上没有交叉支路的电路           |
  - 节点电压法
    适用于平面和非平路电路，在基本节点中选择一个参数节点，对其余的基本节点分配电压变量，并用基尔霍夫电流定律对每个节点列写一个方程
  - 网孔电流法
    适用于平面电路，对每个网孔定义网孔电流并用基尔霍夫电压定律对每个网孔列写一个方程
  - 电源变换
    - 电压源和电阻的串联可变换成电流源和电阻的并联, \[ u = iR \]
    - 戴维南(电压源和电阻的串联)和诺顿(电流源和电阻的并联)可用于简化电源和电阻组成的电路
  - 多个独立源的电路中，叠加原理允许一次用一个电源激励，然后对电压和电流响应求代数和，以确定所有独立源激励时的电压和电流响应，在应用叠加原理时，非独立源永远不能生效

* 运算放大器
  - 理想运放的电压传输特性方程
    \[ u_0 = -V_{cc}, if A(u_p - u_n < -V_{cc})\]
    \[ u_0 = A(u_p - u_n), if -V_{cc} <= A(u_p - u_n) <= V_{cc})\]
    \[ u_0 = V_{cc}, if A(u_p - u_n > V_{cc})\]
  - 运放输出与反相输入之间的反馈路径可以保证运放工作在线性区
  - 工作在线性区的运放由于受到\[V_{cc}\]和A典型值的限制,存在电压的约束。如果线性模型假定是理想的(意味着A假定是无穷大)，则\[ u_p = u_n\]
  - 电流约束进一步构成理想运放模型的特征，因为运放电路的理想输入电阻无穷大，\[ i_p = i_n = 0\]
  - 反相放大器: 输出电压与输入电压反相且成比例的运放电路
  - 求和放大器：输出电压与输入电压之和成比例的运放电路
  - 同相放大器：输出电压与输入电压成比例的运放电路
  - 差分放大器：输出电压与输入电压之差成比例的运放电路
  - 对于差分放大器，两个电压输入可以用来计算共模和差模电压输入，\[u_0 = A_{cm}u_{cm} + A_{dm}u_{dm}\] 其中 \[u_{dm} = u_b - u_a\] \[u_{cm} = (u_a + u_b)/_2\]
  - 共模抑制比 \[CMRR = |A_{dm}/_A_{cm}|\]

* 电感、电容和互感
  - 电感
    - \[u = Ldi/_{di}\]
    - \[i = 1/_L \int_{t_0}^t u \mathrm{d}t + i(t_0)\]
    - \[p = ui = Lidi/_{di}\]
    - \[w = 1/_2Li^2\]
  - 电容
    - \[u = 1/_C \int_{t_0}^t i \mathrm{d}t + u(t_0)\]
    - \[i = Cdu/_{di}\]
    - \[p = ui = Cudu/_{di}\]
    - \[w = 1/_2Cu^2\]
  - 串联与并联
    - 电容串联 \[1/_C_{eq} = 1/_C_1 + 1/_C_2 + 1/_C_3 + ... + 1/_C_n\]
    - 电感串联 \[L_{eq} = L_1 + L_2 + L_3 + ... + L_n\]
    - 电容并联 \[C_{eq} = C_1 + C_2 + L_3 + ... + L_n\]
    - 电感并联 \[1/_L_{eq} = 1/_L_1 + 1/_L_2 + 1/_L_3 + ... + 1/_L_n\]
  - 互感M是一个电路参数，它使一个电路中所感应的电压与另一个电路中的时变电流建立联系，如果线圈绕在磁心体上，则M_{12} = M_{21}
    \[u_1 = L_1di_1/dt + M_{12}di_2/dt\]
    \[u_2 = L_2di_2/dt + M_{21}di_1/dt\]
  - 点约定建立了互感
    /当一个电流的参考方向进入线圈打点端时，则感应在另一线圈上的电压参考极性在打点端为正/
  - 每个线圈自感之间和互感之间的关系 \[M = k\sqrt{L_1L_2}\] k为耦合系数 0<=k<=1
  - 在线性介质中，磁耦合线圈存储的能量与线圈电流和电感有关\[w = 1/2L_1i_1^2 + 1/2L_2i_2^2 \pm Mi_1i_2\]

* 一阶RL和RC电路的响应
  - 一阶电路可以简化为与单个等效电感或电容相连接的戴维南(诺顿)等效电路
  - 固有响应是当存储的能量释放到一个不含独立源的电路时所产生的电流和电压
  - RL电路的时间常数等于等效电感除以等效电感两端看到的戴维南电阻
  - RC电路的时间常数等于等效电容乘以等效电容两端看到的戴维南电阻
  - 阶跃响应是连到路的直流源发生突变所产生的电流和电压
  - RL和RC电路的固有响应或阶跃响应的求解涉及求解有关电流或电压的初值，终值及电路的时间常数
  - 一阶电路中的按序换路是通过划分对应于指定开关位置的时间段来分析的
  - 当戴维南电阻是负的时，将发生无限响应
  - 积分放大器是由理想运算放大器，处在反馈支路的电容和与信号源相串联的电阻组成，电路输出是信号源的积分，在指定范围内应避免运放饱和
* RLC电路的固有响应和阶跃响应
  - 并联RLC电路和串联RLC电路具有相同形式的特征方程，即 \[s^2 + 2as + w_0^2 = 0\], 其中对并联RLC电路,\[a = 1/_{2RC}\]，对串联RLC电路，\[a = R/_{2L}\], \[w_0^2 = 1/_{LC}\]
  - 特征方程的根为 \[s_{1,2} = -a \pm \sqrt_{a^2 - w_0^2}\]
  - 串并联RLC电路的固有响应和阶跃响应的形式取决于\[a^2\]和\[w_0^2\]的取值，分为过阻尼，欠阻尼以及临界阻尼响应3种类型，这些名称描述了耗能元件(R)对电路响应的作用。奈培频率a也与电阻R的大小有关
  - 二阶电路响应特性有三种
    | 电路状态 | 条件            | 响应特性                                           |
    |----------+-----------------+----------------------------------------------------|
    | 过阻尼   | \[a^2 > w_0^2\] | 电压或电流在趋于稳态值的过程中，没有振荡           |
    | 欠阻尼   | \[a^2 < w_0^2\] | 电压或电流在趋于稳态值的过程中，有振荡             |
    | 临界阻尼 | \[a^2 = w_0^2\] | 电压或电流在趋于稳态值的过程中, 处于振荡的临界状态 |
  - 固有响应
    | 电路状态 | 固有响应                                    | 参数方程                          |
    |----------+---------------------------------------------+-----------------------------------|
    | 过阻尼   | \[x(t) = A_1e^{s_1t} + A_2e^{s_2t}\]        | \[x(0) = A_1 + A_2\]              |
    |          |                                             | \[dx/_{dt(0) = A_1s_1 + A_2s_2}\] |
    | 欠阻尼   | \[x(t) = (B_1cosw_dt + B_2cosw_dt)e^{-at}\] | \[x(0) = B_1\]                    |
    |          |                                             | \[dx/_{dt(0) = -aB_1 + w_dB_2}\]  |
    | 临界阻尼 | \[x(t) = (D_1t + D_2)e^{-at}\]              | \[x(0) = D_2\]                    |
    |          |                                             | \[dx/_{dt(0)} = D_1 - aD_2\]      |
  - 阶跃响应
    | 电路状态 | 阶跃响应                                                  | 参数方程                                  |
    |----------+-----------------------------------------------------------+-------------------------------------------|
    | 过阻尼   | \[x(t) = X_f + A_1^{'}e^{s_1t} + A_2^{'}e^{s_2t}\]        | \[x(0) = X_f + A_1^{'} + A_2^{'}\]        |
    |          |                                                           | \[dx/_{dt(0)} = A_1^{'}s_1 + A_2^{'}s_2\] |
    | 欠阻尼   | \[x(t) = X_f + (B_1^{'}cosw_dt + B_2^{'}cosw_dt)e^{-at}\] | \[x(0) = X_f + B_1^{'}\]                  |
    |          |                                                           | \[dx/_{dt(0) = -aB_1^{'} + w_dB_2^{'}}\]  |
    | 临界阻尼 | \[x(t) = X_f + D_1^{'}e^{-at} + D_2^{'}e^{-at}\]          | \[x(0) = X_f + D_2^{'}\]                  |
    |          |                                                           | \[dx/_{dt(0)} = D_1^{'} - aD_2^{'}\]      |
  - 对于3种响应中的未知参数，可以通过计算响应的初值以及初值的一次微分\[dx/_{dt}\]求得
  - 两个理想集成运放电路级联之后，第二个运放的输出与第一个运放的输入信号之间是二次微分方程
  - 简单集成运放电路随着其反馈电容上电荷的积聚，放大倍数会达到饱和，通过在反馈电容上并联一个反馈电阻就可以克服运放的这一局限性
* 正弦稳态分析
  - 正弦电源的一般表达式为 \[u = V_m\cos(\omega t + \phi)\]或 \[i = I_m\cos(\omega t + \phi)\], \[V_m\]和\[I_m\]为最大幅值，\[\omega\]为频率, \[\phi\]为相位角
  - 正弦响应的频率等于驱动电路正弦电源频率, 幅值和相位角通常与电源的不同
  - 时域和频域间的变换:
    - 相量变换(时到频):
      \[V = V_me^{j\phi} = P\{V_m\cos(\omega t + \phi)\}\]
    - 反相量变换(频到时):
      \[P^{-1}\{V_me^{j\phi}\} = \re\{v_me^{j\phi}e^{j\omega t} \} = V_m\cos(\omega t + \phi)\]
  - 在处理正弦变化的信号时，要记住电感两端的电压超前电流90度，电容两端的电流超前电压90度
  - 频域阻抗与时域电阻、电感和电容一样，对电阻、电感、电容来说，电流相量和电压相量间具有关系式\[V = ZI\], 其中I的参考方向按习惯规定，阻抗的倒数称为导纳(Y), 即\[V=I/Y\]
  - KVL, LCL, 串联阻抗之和，并联阻抗之和，分压分流定理，节点电压和网孔电流法，电源变换、戴维南和诺顿等效变换，以上也适用于频域下的正弦稳态分析
  - 线性变压器是两个线圈缠绕在同一个非磁性物质上构成的耦合器件，线圈的次级电路在初级上产生的阻抗称为反射阻抗，反之亦然。从线性变压器的初级看过去的反射阻抗为次级电路的自阻抗经过变换得到的，即\[(wM/|Z_{22}|)^2\]
  - 理想变压器是线性变压器，具有如下特性：为最佳耦合(K=1)，每个线圈的自感均为无穷大，以及线圈损耗为0，电路的行为受变压器的匝数比\[a = N_2/_{N_1}\]，且有如下公式:
    \[V1/_{N_1} = \pm V2/_{N_2}\] 和 \[N_1I_1 = \pm N_2I_2\]
  - 阻抗
    | 元件 | 阻抗(Z)  | 电抗  | 导纳(Y)  | 电纳  |
    |------+----------+-------+----------+-------|
    | 电阻 | R        |       | G        |       |
    | 电容 | j(-1/wC) | -1/wC | jwC      | wC    |
    | 电感 | jwL      | wL    | j(-1/wL) | -1/wL |
* 正弦稳态功率计算
  - 瞬时功率是瞬时端电压和端电流的积，即\[p = \pm ui\]。当电流的参数方向是从电压的正端流向负端时用正号。瞬时功率的频率是电压(或电流)频率的两倍。
  - 平均功率是瞬时功率在一个周期上的平均值。它是电能转变为其他形式能量或者其他形式能量转变为电流的功率。也称平均功率为实功率，在无源符号约定下，平均功率表示为
    \[P = 1/_2 V_mI_m\cos(\theta_u - \theta_i) = V_{eff}I_{eff}\cos(\theta_u - \theta_i)\]
  - 无功功率是在电感磁功和电源之间，或电容磁场和电源之间相互传递的电能，无功功率不会转变为其他形式的能量，在无源符号约定下，无功功率表示为
    \[Q = 1/_2 V_mI_m\sin(\theta_u - \theta_i) = V_{eff}I_{eff}\sin(\theta_u - \theta_i)\]
    平均功率和无功功率都可以用电流和电压的峰值\[(V_m, I_m)\]或者有效都\[(V_eff, I_eff)\]表示，有效值和均方根值是同一物理值的不同称呼
  - 功率因数是电压和电流相位角差的余弦函数 \[pf = cos(\theta_u - \theta_i)\]，在功率因数的描述中加入了滞后和超前，以确定负载是感性还是容性
  - 无功因数是电压和电流相位角之差的正弦函数 \[pf = sin(\theta_u - \theta_i)\]
  - 复功率是平均功率和无载功率的复数和，即\[S = P + jQ = 1/_2 VI^{*} = V_{eff}I_{eff}^2 = I_{eff}^2 Z = V_{eff}^2 / _{Z^{*}}\]
  - 视在功率是复功率的模 \[|S| = \sqrt{P^2 + Q^2}\]
  - w(瓦特)是平均功率和瞬时功率的单位。VAR(乏)是无功功率的单位。VA(伏安)是复功率和视在功率的单位
  - 最大功率传输。当负载阻抗等于从负载阻抗两端看过去的等效戴维南阻抗的共轭复数时，正弦稳态电流传递给负载的平均功率最大
* 平衡三相电路
  - 分析平衡三相电路时，第一步是将所有的\[\Delta\]形连接变换为Y形连接，使电路变为Y-Y形结构
  - 单相等效电路用来分析Y-Y结构单相的线电流和电压，通常选择a相
  - 已知a相等效电路的线电流和电压，在下面事实的基础上可以求出平衡三相电路的任何电流和电压
    - b相和c相的电流和电压与a相的电流和电压想同，除了相位相差120度，在正相序电路中，b相的值滞后于a相120，c相的值超前于a相120，在负相序中，相位差相反
    - 线电压与相电压相位相差\[\pm 30^0\]，正负号分别取决于相序为正还是负
    - 在Y-Y电路中，线电压的幅值是相电压幅值的\[\sqrt{3}\]倍
    - 在电源和负载为\[\Delta\]连接的系统中，线电流和相电流的相差\[\pm 30^o\]。正负号分别取决于相序为正还是负
    - 在电源和负载为\[\Delta\]连接的系统中，线电流的幅值是相电流幅值的\[\sqrt{3}\]倍
  - 平衡三相电路的总瞬时功率为常数\[1.5V_mI_mpf\]
  - 瓦特计通过将电流线圈与负载串联，电位线圈与负载并联，测出发送到负载上的平均功率
  - 平衡三相电路中的总平均功率可以通过连接在电路不同相上的两个瓦特计的读数相加得到
* 拉普拉斯变换简介
  - 拉氏变换是时域到频域的变换工具，其定义如下:
    \[F(s) = \int_{0}^{\infty}f(t)e^{-st}dt\]
  - 阶跃函数Ku(t)表示一个函数在时域变量经过间断点时，函数值从一个值跳变到另一个值。K是跳变的幅度，若K=1，则Ku(t)就是单位阶跃函数
  - 冲激函数\[K\delta(t)\]定义是\[\int_{-\infty}^{\infty}K\delta(t)dt = K\]其中\[\delta(t) = 0\]当\[t\neq0\]
    K为冲激强度，若K=1，则\[K\delta(t)\]为单位冲激函数
  - 函数变换是一些典型函数的拉氏变换，算子变换定义了拉氏变换的普通数学特性
  - *在线性集总电路中，F(s)是s的分式*
  - 若F(s)是有理真分式，则逆变换可以通过部分分式法求得
  - 若F(s)是假分式，则可先将其化为多项式与有理真分式之和，进而再求得逆变换
  - F(s)可以表示成两个因式相乘形式多项式的比，分母的根为极点，在s平面上用\[\times\]表示，分子的根为零点，在s平面上用\[\bigcirc\]表示
  - 初值定理 \[\lim_{t->0^+}f(t) = \lim_{s->\infty}sF(s)\]，定理中假设f(t)不包含冲激函数
  - 终值定理 \[\lim_{t->\infty}f(t) = \lim_{s->0^+}sF(s)\]，当F(s)所有极点(除原点处的一阶极点)位于s平面的左半面时，该理理成立
  - 初值和终值定理可由s域表达式求出f(t)的初值和终值
* 拉氏变换在电路分析中的应用
  - *将元件伏安关系进行拉氏变换，可以将每个电路元件表示成s域的等效电路*
    - 电阻: \[V = RI\]
    - 电感: \[V = sLI - LI_0\]
    - 电容: \[V = 1/_{sC}I + V_0/_s\]
  - 将电路的每个元件都用s域等铲电路来代替，就可以对电路进行s域分析了。用电阻电路分析方法写出电路的代数方程就可以对所得的电路进行分析
  - 当已经电路初始条件时，对线性集总电路用s域分析法最适合求暂态响应，因为这种分法将微分方程化为了代数方程，所以对含有多网孔和多节点方程问题的解，可以先变换后求解
  - 转移函数是电路输出与输入的s域的比值，即\[H(s) = Y(s)/_{X(s)}\]
  - H(s)和X(s)的积可以展开为部分分式的形式，H(s)和X(s)的每个极点都对应展开式中的一项。展开式中H(s)的极点对应的项为暂态响应分量，X(s)极点对应的项为稳态响应分量
  - 如果电路的激励为单位冲激\[x(t) = \delta(t)\]，那么电路的响应就等于转移函数的拉氏逆变换\[y(t) = F^{-1}\{H(s)\} = h(t)\]
  - 对非时变电路，如果输入延时a秒，那么输出函数也将延时a秒
  - 电路的输出y(t)可以通过输入函数x(t)与电路单位冲激响应h(t)的卷积来计算
    \[y(t) = h(t) * x(t) = \int_0^th(\lambda)x(t-\lambda)d\lambda = x(t) * h(t) = \int_0^tx(\lambda)h(t-\lambda)d\lambda\]
  - 可以用电路的转移函数来计算正弦电源激励下电路的稳态响应，方法是将H(s)中的s用\[j\omega\]代替，然后用复数表示幅值和相位，如果
    \[x(t) = \cos(\omegat + \phi)\]
    \[H(j\omega) = |H(j\omega)|e^{j\theta\omega}\]
    则 \[y_{ss}(t) = A|H(j\omega)|\cos[\omega t + \phi + \theta(\omega)]\]
  - 拉氏变换分析法能正确计算开关操作和冲激电源引起的冲激电流与电压，但必须保证在\[t = 0^-\]时刻的初始条件基础上建立s域等效电路。也就是说，以开关动作之前电路所处状态下的条件为基础，建立s域的等效电路
* 选频电路
  - 选频电路(或称为滤波器)能使某些频率的信号到达输出端，而削弱其他频率的信号，使它们不能到达输出端。通带包含可通过信号的频率，阻带包含被削弱信号的频率
  - 截止频率\[\omega_c\]，是频率轴上定义的一个点，它将通带和阻带分开，在截止频率处，转移函数的幅值等于最大值的\[1/_{\sqrt{2}}\]
  - 低通滤波器，使频率低于\[\omega_c\]的信号通过，削弱频率高于\[\omega_c\]的信号，任何具有下列形式的转移函数都可作为低通滤波器\[H(s) = \omega_c/_{s + \omega_c}\]
  - 高通滤波器，使频率高于\[\omega_c\]的信号通过，削弱频率低于\[\omega_c\]的信号，任何具有下列形式的转移函数都可作为高通滤波器\[H(s) = s/_{s + \omega_c}\]
  - 带通滤波器和带阻滤波器都具有两个截止频率\[\omega_{c1}\]，\[\omega_{c2}\]。这些滤波器的特性由中心频率\[\omega_{0}\]，带宽\[\beta\]和品质因数Q来描述
    \[\omega_{0} = \sqrt{\omega_{c1} * \omega_{c2}}\]
    \[\beta = \omega_{c2} - \omega_{c1}\]
    \[Q = \omega_{0}/\beta\]
  - 带通滤波器将处于通带之内(即\[\omega_{c1}\]和\[\omega_{c2}\])的信号通过，而削弱通带以外的信号。任何具有下面形式转移函数的系统均可作为带通滤波器
    \[H(s) = \beta_s / _{s^2 + \beta_s + \omega_0^2}\]
  - 无源滤波器的输出端加负载会改变通带的幅值和位置，从而改变滤波器特性，以非零内阻的电源代替理想电压源也会改变通带的幅值和位置，从而改变电路的滤波特性
* 有源滤波器电路
  - 有源滤波器由理想运放、电阻和电容元件组成，可以构成低通、高通、带通和带阻滤波器，有源滤波器克服了无源滤波器的许多缺点
  - 原型低通滤波器，其元件值为\[R_1 = R_2 = 1ohm\], \[C = 1F\]，通带放大系数为1，截止频率为1rad/s，原型高通滤波器也具有同样的元件、放大系数和截止频率
  - 幅值比例变换法，在不改变电路频率响应的情况下，改变元件的值，若幅值比例系数为\[k_m\]，则比例放大前后元件值之间的关系为
    \[R^{'} = k_mR\], \[L^{'} = k_mL\], \[C^{'} = C/_{k_m}\]
  - 频值比例变换法，在不改变频率响应形状的情况下，可以将电路频率响应的位置移到另一个频率区域，若频率比例因数为\[k_f\]，则比例变换后各元件间的关系为
    \[R^{'} = R\], \[L^{'} = L/_{k_f}\], \[C^{'} = C/_{k_f}\]
  - 元件值可以通过幅值和频率比例变换进行比例变换，比例变换前后关系为
    \[R^{'} = k_mR\], \[L^{'} = (k_m/_{k_f})L\], \[C^{'} = C/_{k_mk_f}\]
  - 有源高通和低通滤波器的设计都可以从原型滤波器电路开始，比例变换法可以用于将频率响应移动到要求的截止频率位置上，元件参数应选市场上可以买到的元件值
  - 低通滤波器、高通滤波器和反相放大器通过级数就可以构成有源宽带带通滤波器，其中低通滤波器具有较高的截止频率，高通滤波器具有较低的频率，反相放大器用来完成通带放大系数非1的放大。用这种方式构成的带通滤波器必须是宽带滤波器\[(\omega_{c2} = \omega_{c1})\]，以便各级联单元可以相互间独立确定
  - 有源宽带带阻滤波器可以由低通滤波器、高通滤波器和积分放大器等单元的并联构成，低通滤波器的截止频率是带阻滤波器的较低截止频率。高通滤波器的截止频率是带阻滤波器的较高截止频率，其输出与积分放大器的输入相连，该积分放大器产生通带内的放大倍数，用这种方法构成的带阻滤波器必须是宽带滤波器\[(\omega_{c2} = \omega_{c1})\]，这样才能使得低通和高通滤波器之间实现相互独立的设计
  - 高阶有源滤波器的转移函数有多个极点，使得从通带向阻带的过渡非常快，因而更加接近于理想的频率响应
  - 截止频率为1rad/s的n阶巴特沃思低通滤波器的转移函数可以由下面的方程确定
    \[H(s)H(-s) = 1/_{1 + (-1)^ns^2n}\]
    - 找到分母多项式的根
    - 将左半面的根赋给H(s)
    - 写出一阶和二阶因式的乘积作为H(s)的分母
  - 设计巴特沃思滤波器的基本问题是确定滤波器的阶数，滤波器通常根据4个量\[A_p\]，\[\omega_p\]，\[A_s\]和\[\omega_s\]定义过渡域的斜度，根据这几个量可以计算出最小整数
  - 二阶低通运放滤波器具有\[1ohm\]电阻值和可以选择的电容，能产生巴特沃思多项式的因式，这种滤波器的级联可以构成偶阶的巴特沃思低通滤波器，若加一个级联的原型低通运放滤波器就会构成奇阶的巴特沃思低通滤波器
  - 二阶高通运放滤波器具有1F电容和可以选择的电阻，能构成巴特沃思多项式的因式，这种滤波器的级联可以构成偶阶的巴特沃思高通滤波器，若再级联一个原型高通运放滤波器就构成奇阶巴特沃思高通运放滤波器
  - 在设计中，无论是高通滤波器还是低通滤波器，都可以用频率比例变换将截止频率从1rad/s处移走，都可以用频率比例变换和幅值比例变换选择实际的元件值，级联一个反相放大器后将能产生非单位的通带放大系数
  - 将巴特沃思高通和低通滤波器进行级联，可以构成任意阶的巴特沃思带通滤波器，将巴特沃思高通和低通滤波器并联后再级联求和放大器，可以构成n阶的巴特沃思带阻滤波器
  - 若需要一个高品质因数、窄带的带通或带阻滤波器，那么级联和并联方法并不适用。一般情况下，电容值要根据市场情况选定，而电阻值则根据设计方程确定
* 傅里叶级数
  - 周期函数是这样一种函数，它每隔T秒循环一次
  - 周期是一个周期函数循环重复的 *最短时间间隔* (T)
  - 傅里叶级数是一个用来表示周期函数的无穷级数。它由常数项和无限多正交的正弦项和余弦项组成
  - 基频是由基波周期(\[f_0 = 1/T\])或\[\omega_0 = 2\pi f_0\]
  - 谐波频率是基频的整数倍
  - 傅里叶系数是常数项，级数中每个正弦项和余弦项的系数也是常数
  - 5种类型的对称可以用来简化傅里叶系数计算:
    - 偶对称，其傅里叶系数的所有正弦项为0
    - 奇对称，其傅里叶系数的所有常数项和余弦项为0
    - 半波对称，其傅里叶级数的所有偶次谐波为0
    - 四分之一波对称、半波对称、偶对称，其傅里叶级数中只含有奇次谐波的余弦项
    - 四分之一波对称、半波对称、奇对称，其傅里叶级数中只含有奇次谐波的正弦项
  - 在三角函数形式的傅里叶级数中，用正弦项和余弦项之和表示的各次谐波被合并成\[A_n\cos(n\omega_0t - \theta_n)\]形式的单一项
  - 对于稳态响应，求响应信号的傅里叶级数，可以先求输入信号的各个分量的响应。然后将单个响应相加，从而得到响应信号的傅里叶级数。输入信号傅里叶级数中单个项的响应通过频域或s域分析求得
  - 没有计算机的帮助，很难得到响应信号的波形。有时，可以利用电路的频响(或滤波)特性确定输出波形与输入波形的匹配程度
  - 只有同频率的谐波相互作用产生平均功率。总平均功率是各个频率对应的平均功率之和
  - 周期函数的均方根值可以由傅里叶系数估计得到
  - 利用欧拉公式，可以将正弦项和余弦项用相应的指数恒等式表示，傅里叶级数可以用指数形式表式
  - 当系统的激励为周期信号时，可以利用傅里叶级数预测该系统的稳态响应
* 傅里叶变换
  - 傅里叶变换给出了一个非周期时域函数的频域描述。根据时域信号的类型，可以用下列三种方法之一求信号的傅里叶变换
    - 如果时域信号是一个有限延迟脉冲，则可以利用积分定义求其傅里叶变换
    - 如果f(t)的单边拉氏变换存在，且F(s)的所有极点位于s平面的左半部分，则可以用F(s)求F(w)
    - 如果f(t)是一个常数、符号函数、阶跃函数域正弦函数，则该函数的傅里叶变换可以通过极限得到
  - 响应信号y(t)的傅里叶变换为 \[Y(\omega) = X(\omega)H(\omega)\]
    X(w)是输入信号x(t)的傅里叶变换， *H(w)是传递函数H(s)在s=jw处的值*
  - 因为傅里叶变换对正时间函数和负时间函数都适用，所以傅里叶变换也适合解决由\[t = -\infty\]时开始的事件描述的问题，与此相反，单边拉氏变换适合解决由初始条件和t>0时发生的事件描述的问题
  - 傅里叶变换的平方值是频域内能量密度(焦耳/赫兹)的量度(帕塞瓦尔定理)，因此，傅里叶变换将f(t)包含的总能量的一部分与一个焦耳特定频率带联系起来
* 双端口网络
  - 双端口网络模型用来根据输入、输出端的电压和电流描述电路的特性
  - 双端口网络模型所描述的电路仅限于
    - 端口间的电路无独立源
    - 电路内无储能
    - 流入流出端口的电流相等
    - 输入和输出端口间无外部连接
